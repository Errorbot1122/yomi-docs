Verbiage
========

These are a list of *terms* that our *Community* uses that may be confusing to newcomers:


General
-------

.. _yverb-yomi:

* :ref:`🔗<yverb-yomi>` **YOMI:** An abbration for *"Your Only Move is"*. Usually refers to the game `Your Only Move is Hustle <https://store.steampowered.com/app/2212330/Your_Only_Move_Is_HUSTLE/>`_.

.. _yverb-yomih:

* :ref:`🔗<yverb-yomih>` **YOMIH, YOMI Hustle:** An abbreviation for the game `Your Only Move is Hustle <https://store.steampowered.com/app/2212330/Your_Only_Move_Is_HUSTLE/>`_.

.. _yverb-ivy:

* :ref:`🔗<yverb-ivy>` **Ivy, Ian:** Slang for the creator of `Your Only Move is Hustle <https://store.steampowered.com/app/2212330/Your_Only_Move_Is_HUSTLE/>`_, `Ivy Sly (Ian Sly) <https://ivysly.com/about>`_.

.. _yverb-hustle:

* :ref:`🔗<yverb-hustle>` **Hustle:** Sometimes the name of the common move across all :ref:`YOMIH <yverb-yomih>` Characters, sometimes an abbreviation of the `game <https://store.steampowered.com/app/2212330/Your_Only_Move_Is_HUSTLE/>`_.

.. _yverb-qol:

* :ref:`🔗<yverb-qol>` **QOL:** Short for "Quality of Life", used to denote mods/features that make the game easier to play.

.. _yverb-charstateeditor:

* :ref:`🔗<yverb-charstateeditor>` **CharStateEditor, "hi :)", "1738 ayy im like hey whats up hello":** The *somewhat*-internal name for the :ref:`Characters State Editor <ydoc-tutorial-state-editor>`.
