.. _class_characterstate_property_iasa_at:


:badge-export:`@export` 


.. rst-class:: classref-property

:ref:`int<class_int>` **iasa\_at** = ``- 1`` :ref:`🔗<class_characterstate_property_iasa_at>`

The tick number (0 being when the state starts) when the player will get their next interrupt turn.

Negative numbers will be converted during  :ref:`^init()<class_characterstate_method_init>`  to be that many ticks before  :ref:`^ObjectState.anim_length<class_objectstate_property_anim_length>` , with the default of -1 being the tick this state naturally ends.

Has no effect if  :ref:`^ObjectState.endless<class_objectstate_property_endless>`  is ``true``, or  :ref:`^interrupt_into_string<class_characterstate_property_interrupt_into_string>`  is empty.

The tick number (0 being when the state starts) when the player will get their next interrupt turn if one of this state's hitboxes hits something during this state's use.

Negative values have no effect.

Has no effect if  :ref:`^ObjectState.endless<class_objectstate_property_endless>`  is ``true``, or  :ref:`^interrupt_into_string<class_characterstate_property_interrupt_into_string>`  is empty.
