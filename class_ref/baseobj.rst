BaseObj
=======

**Inherits:** :ref:`class_node2d` **<** :ref:`class_canvasitem` **<** :ref:`class_node` **<** :ref:`class_object`

**Inherited By:** :ref:`class_baseprojectile`, :ref:`class_fighter`

Class containing functionality that both  :ref:`^Fighter<class_fighter>`  and  :ref:`^BaseProjectile<class_baseprojectile>`  use.

**Note:** This class is missing functionality that is added by  :ref:`^Fighter<class_fighter>`  and  :ref:`^BaseProjectile<class_baseprojectile>` .  Despite its name, all game objects that are not  :ref:`^Fighter<class_fighter>` s should extend  :ref:`^BaseProjectile<class_baseprojectile>` , not  :ref:`^BaseObj<class_baseobj>`  directly.
