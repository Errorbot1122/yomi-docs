BaseProjectile
==============

**Inherits:** :ref:`class_baseobj` **<** :ref:`class_node2d` **<** :ref:`class_canvasitem` **<** :ref:`class_node` **<** :ref:`class_object`



Class for any game object that is not a  :ref:`^Fighter<class_fighter>` .

**Note:** Despite its name, objects extending  :ref:`^BaseProjectile<class_baseprojectile>`  do not have to be standard projectiles.   :ref:`^BaseProjectile<class_baseprojectile>`  is flexible enough and meant to be used for ANY non- :ref:`^Fighter<class_fighter>`  game object.
